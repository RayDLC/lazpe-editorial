## Steps to run this project

1. Clone the repository

    `git clone https://github.com/RayDLC/lazpe-editorial.git`

2. Install the package

    `pnpm intall`

3. Initialize the server in localhost

    `ng serve`
4. Open the following url

    `localhost:4200`

## Stack used 

+ Angular v16
+ Tailwind
+ Angular Material
+ Package Manger (pnpm)
